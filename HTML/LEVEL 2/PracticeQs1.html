Common HTML Entities

  • **Ampersand**: & → &amp;
  • **Less Than**: < → &lt;
  • **Greater Than**: > → &gt;
  • **Double Quote**: " → &quot;
  • **Single Quote (Apostrophe)**: ' → &apos;

Mathematical Symbols

  • **Plus**: + → &plus;
  • **Minus**: − → &minus;
  • **Multiplication**: × → &times;
  • **Division**: ÷ → &divide;
  • **Plus-Minus**: ± → &plusmn;
  • **Square Root**: √ → &radic;
  • **Infinity**: ∞ → &infin;

Currency Symbols

  • **Dollar**: $ → &dollar;
  • **Euro**: € → &euro;
  • **Pound**: £ → &pound;
  • **Yen**: ¥ → &yen;

Arrows

  • **Left Arrow**: ← → &larr;
  • **Up Arrow**: ↑ → &uarr;
  • **Right Arrow**: → → &rarr;
  • **Down Arrow**: ↓ → &darr;

Greek Letters

  • **Alpha**: α → &alpha;
  • **Beta**: β → &beta;
  • **Gamma**: γ → &gamma;
  • **Delta**: δ → &delta;
  • **Pi**: π → &pi;

Miscellaneous Symbols

  • **Copyright**: © → &copy;
  • **Registered Trademark**: ® → &reg;
  • **Trademark**: ™ → &trade;
  • **Degree**: ° → &deg;
